# Los Molinos Pet Food

Sitio web oficial de Los Molinos Pet Food, presentando nuestras líneas de alimentos balanceados premium para mascotas: Deleita, Winy, Ducho y Rabito.

## Descripción

Este proyecto es una landing page moderna y responsiva que muestra:
- Información institucional (Nuestra Fábrica).
- Líneas de productos (Super Premium, Premium, Estándar).
- Puntos de venta interactivos.
- Catálogo de productos con carrito de compras (funcionalidad frontend).
- Formulario de contacto para mayoristas.

## Tecnologías

- **HTML5** semántico.
- **CSS3** moderno (Flexbox, Grid, Variables CSS).
- **JavaScript** (ES6+) para interactividad (Carrito, Carrusel, Tabs).
- **Font Awesome** para iconos.
- **Google Fonts** (Playfair Display, Raleway).
